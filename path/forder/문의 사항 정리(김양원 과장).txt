궁금증 정리
로컬 개발환경 문의
- 백엔드 소스에 보면 default로 설정된 로컬 개발환경이 있는데 이 환경을 계정과 접속 정보만 변경하여 사용 가능한지?

모니터링 방법?
- LG CNS에서는 데이터독을 쓰는것으로 알고 있는데, 데이터독을 모니터링 서비스로 사용 가능한지 확인 부탁드립니다. 

백엔드 아키텍처에 Redis SSO의 사용 여부
- 이번 운항통제 시스템에 Redis를 사용해야 하는지의 여부 확인 요청 드립니다.

GIT 저장소는 어디이며, 신청여부
- Git 저장소는 따로 신청을 해야하는지 확인 요청 드립니다.

FrontEnd에서의 apigee 사용 여부
- 제 생각으로는 클라이언트는 프론트엔드의 ALB URL을 호출하고, 프론트 엔드 URL을 기반으로 apigee에 등록된 백엔드를 호출하는 로직으로 생각이 됩니다만, 맞는지 확인 부탁드립니다.

암호화대상 컬럼에 대한 문의
- 현재 하는 프로젝트에 암호화 대상 컬럼이 존재 하는지 확인 부탁 드립니다.
